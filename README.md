# douyinquick
一款抖音解析工具app
1、权限：悬浮窗、联网、访问储存
2、使用：抖音app（国内版）；打开软件，给予权限（重启app），自动跳转到抖音，找到自己喜欢的视频->分享->复制链接,之后按提示操作。视频下载在DouyinQuick目录下。
